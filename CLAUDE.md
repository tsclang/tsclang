# TSClang — инструкции для Claude

## О проекте

TSClang — компилятор TypeScript-подобного языка (расширение `.tsc`) в C с автогенерацией `CMakeLists.txt`.

- CLI: `tsclang` (npm-пакет)
- Таргеты: desktop (libuv) и embedded (AVR, без heap-async)
- Текущий статус: стадия дизайна, кода компилятора ещё нет

## Ключевые файлы

| Файл | Содержимое |
|------|-----------|
| `CONCEPT.md` | Полная спецификация языка — читать перед любой работой |
| `PROBLEMS.md` | Выявленные проблемы дизайна |
| `EDIT.md` | Внешний анализ проблем (дополняет PROBLEMS.md) |

## Язык общения

Общаться на **русском**. Комментарии в коде — на **английском**.

## Дизайн-философия

См. `CONCEPT.md` — разделы **Зачем** и **Дизайн-философия**.

## Стиль работы

- Перед предложением изменений — читать CONCEPT.md в нужной части
- Не добавлять фичи сверх запрошенного
- Изменения в дизайне фиксировать в CONCEPT.md, проблемы — в PROBLEMS.md
- Код компилятора будет на TypeScript (Node.js)
- Тесты — формат: входной `.tsc` → ожидаемый C output или ошибка компилятора

## Структура проекта (планируемая)

```
src/          — исходный код компилятора (TypeScript)
doc/          — тестовый корпус (примеры .tsc → C)
log/          — логи решений по компонентам
bin/index.js  — точка входа CLI
```
