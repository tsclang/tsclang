# Дефолтные параметры

[← Вверх](./index.md) | [Предыдущий ←](./overload.md)

---

Параметры со значениями по умолчанию. Работают для функций, методов и конструкторов. На callsite компилятор подставляет дефолтное значение в C-output.

## Синтаксис

```typescript
function add(x: i32, y: i32 = 10): i32 {
    return x + y;
}

console.log(add(5));      // 15 — y подставляется = 10
console.log(add(5, 20));  // 25 — y = 20
```

**C-output:**

```c
int32_t add_i32_i32(int32_t x, int32_t y) {
    return x + y;
}

// callsite:
printf("%d\n", add_i32_i32(5, 10));   // подстановка дефолта
printf("%d\n", add_i32_i32(5, 20));   // явное значение
```

Дефолтное значение подставляется **на callsite** — сама функция в C не имеет дефолтных значений параметров.

## Несколько дефолтных параметров

```typescript
function calc(x: i32, y: i32 = 2, z: i32 = 3): i32 {
    return x + y + z;
}

console.log(calc(1));        // 6  — y=2, z=3
console.log(calc(1, 10));    // 14 — y=10, z=3
console.log(calc(1, 10, 20)); // 31 — y=10, z=20
```

**C-output:**

```c
int32_t calc_i32_i32_i32(int32_t x, int32_t y, int32_t z) {
    return x + y + z;
}

printf("%d\n", calc_i32_i32_i32(1, 2, 3));
printf("%d\n", calc_i32_i32_i32(1, 10, 3));
printf("%d\n", calc_i32_i32_i32(1, 10, 20));
```

## Дефолтные параметры в методах

```typescript
class Printer {
    print(text: string, times: i32 = 1): void {
        for (let i = 0; i < times; i++) {
            console.log(text);
        }
    }
}

printer.print("hi");      // times=1
printer.print("hi", 3);   // times=3
```

## Правила

- Дефолтные параметры должны быть **в конце** списка параметров
- Дефолтное значение — **константа или литерал**, не выражение с побочными эффектами
- Запрещено иметь overload, сигнатура которого совпадает с вызовом другого overload при подстановке дефолтных значений

## Взаимодействие с перегрузкой

Комбинация дефолтных параметров и перегрузки может привести к неоднозначности — это ошибка компилятора:

```typescript
function foo(x: i32, y: i32 = 0): void { /* ... */ }
function foo(x: i32): void { /* ... */ }
// ❌ ошибка: ambiguous overload — foo(x: i32) совпадает с foo(x: i32, y: i32 = 0)
```

Если нужна перегрузка по количеству параметров — используйте дефолтные параметры **или** overload, но не оба одновременно для одной и той же сигнатуры.

---

## Ошибки

| Ошибка | Причина |
|--------|---------|
| `default parameter must be at end of parameter list` | Дефолтный параметр перед обязательным |
| `default value must be constant or literal` | Выражение с побочными эффектами в дефолтном значении |
| `ambiguous overload with default parameters` | Перегрузка дублирует вызов с дефолтными параметрами |
