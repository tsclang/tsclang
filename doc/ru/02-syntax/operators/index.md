## Общие правила

- **Нет неявного приведения типов** — `==` и `===` ведут себя одинаково
- **Ownership** — сложные типы перемещаются (move) при присваивании, примитивы копируются
- **`const`** — нельзя переприсвоить, нельзя использовать левосторонние операторы (`++`, `+=` и т.д.)
