# Битовые операторы

[← Вверх](./index.md) | [Следующий →](./optional.md) | [Предыдущий ←](./logical.md)

---

Побитовые операции над целыми числами. Работают с типами `i8`..`i64`, `u8`..`u64`, `usize`. Применение к `f32`/`f64` или `string` — ошибка компилятора.

## Операторы

| Оператор | Описание | Пример |
|----------|----------|--------|
| `&` | Побитовое И | `a & b` |
| `\|` | Побитовое ИЛИ | `a \| b` |
| `^` | Побитовое XOR | `a ^ b` |
| `~` | Побитовое НЕ (унарный) | `~a` |
| `<<` | Сдвиг влево | `a << n` |
| `>>` | Сдвиг вправо (знаковый) | `a >> n` |
| `>>>` | Сдвиг вправо (беззнаковый) | `a >>> n` |

---

## Побитовое И (`&`), ИЛИ (`|`), XOR (`^`)

Побитовые операции выполняются над каждым битом:

```typescript
const a: u8 = 0b1100_1010;
const b: u8 = 0b1010_0110;

const and: u8 = a & b;    // 0b1000_0010  — бит 1 только если оба 1
const or: u8  = a | b;    // 0b1110_1110  — бит 1 если хотя бы один 1
const xor: u8 = a ^ b;    // 0b0110_1100  — бит 1 если ровно один 1
```

Типичные сценарии:

```typescript
// маска — извлечь конкретные биты
const flags: u32 = 0xFF00;
const lower: u32 = flags & 0x00FF;     // 0x0000 — нижний байт
const upper: u32 = flags & 0xFF00;     // 0xFF00 — верхний байт

// установить бит
let ctrl: u32 = 0x00;
ctrl |= 0x01;          // установить бит 0
ctrl |= 0x80;          // установить бит 7

// сбросить бит
ctrl &= ~0x80;         // сбросить бит 7

// инвертировать бит
ctrl ^= 0x01;          // инвертировать бит 0
```

## Побитовое НЕ (`~`)

Инвертирует все биты (унарный):

```typescript
const a: u8 = 0b0000_1111;
const not: u8 = ~a;     // 0b1111_0000

// типичный паттерн: сбросить бит через &
let flags: u32 = 0xFF;
flags &= ~0x0F;         // 0xF0 — сбросить младшие 4 бита
```

## Сдвиги

### `<<` — сдвиг влево

Сдвигает биты влево, заполняя нулями. Эквивалент умножения на `2^n`:

```typescript
const val: u32 = 1;
const shifted: u32 = val << 4;    // 16 — 1 * 2^4
```

### `>>` — знаковый сдвиг вправо

Сохраняет знаковый бит. Для знаковых типов заполняет старшие биты знаком:

```typescript
const neg: i32 = -8;
const result: i32 = neg >> 1;     // -4 — знаковый бит сохранён

const pos: u32 = 16;
const div: u32 = pos >> 2;        // 4 — 16 / 2^2
```

### `>>>` — беззнаковый сдвиг вправо

Всегда заполняет старшие биты нулями, даже для знаковых типов:

```typescript
const neg: i32 = -1;              // 0xFFFFFFFF
const result: u32 = neg >>> 4;    // 0x0FFFFFFF — нули вместо знака

// извлечение байтов из u32
const value: u32 = 0xAABBCCDD;
const byte0: u8 = (value & 0xFF) as u8;           // 0xDD
const byte1: u8 = ((value >> 8) & 0xFF) as u8;    // 0xCC
const byte2: u8 = ((value >> 16) & 0xFF) as u8;   // 0xBB
const byte3: u8 = ((value >>> 24) & 0xFF) as u8;  // 0xAA
```

## C-output

Битовые операции транслируются напрямую — один к одному:

```c
// const and: u8 = a & b;
uint8_t and = a & b;

// const or: u8 = a | b;
uint8_t or = a | b;

// const xor: u8 = a ^ b;
uint8_t xor = a ^ b;

// const not: u8 = ~a;
uint8_t not = ~a;

// const shifted: u32 = val << 4;
uint32_t shifted = val << 4;

// const result: i32 = neg >> 1;
int32_t result = neg >> 1;

// const r: u32 = neg >>> 4;
uint32_t r = (uint32_t)neg >> 4;
```

## Ошибки

| Ошибка | Причина |
|--------|---------|
| `bitwise operator & not defined for f64` | Битовые операции с float |
| `bitwise operator & not defined for string` | Битовые операции со строками |
