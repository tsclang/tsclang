# Учебник TSClang

Этот учебник научит вас писать программы на TSClang шаг за шагом. Мы не будем перечислять все возможности языка — для этого есть [документация](../doc/ru/index.md). Здесь мы решаем задачи и осваиваем инструменты по мере необходимости.

## Для кого этот учебник

Вы знакомы с TypeScript или JavaScript и хотите:
- компилировать код в читаемый C вместо байт-кода VM;
- запускать программы на микроконтроллерах (AVR, WASM, retro-консоли);
- работать с безопасной моделью памяти без сборщика мусора.

## Как работать с учебником

Каждая глава содержит:
1. **Цель** — что мы построим.
2. **Новые понятия** — краткий список.
3. **Пошаговое руководство** — пишем код блок за блоком.
4. **Полный исходник** — готовый файл для копирования.
5. **Компиляция и запуск** — точные команды.
6. **Упражнения** — задания для самостоятельной работы.

> **Совет:** не копируйте код целиком. Набирайте руками — так лучше запоминается синтаксис и ошибки компилятора.

## Требования

- Node.js 18+
- `tsclang` установлен глобально: `npm install -g tsclang`
- GCC или Clang (для запуска desktop-target)
- (опционально) AVR-GCC и avrdude (для главы 07)
